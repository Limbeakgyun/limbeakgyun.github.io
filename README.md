# Lim Beakgyun GitHub Pages Starter

This folder contains a static GitHub Pages site.

## Publish

1. Create or open the repository `limbeakgyun.github.io`.
2. Copy the files in this folder to the repository root.
3. Commit and push to GitHub.
4. In the repository settings, confirm GitHub Pages is serving from the root of the default branch.

## Files

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`
