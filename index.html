소개

빠르게 손보고 천천히 채워 넣는 개인 사이트.

구조를 단순하게 유지해서 소개 문구, 프로젝트 설명, 링크만 바꿔도 전체 인상을 다시 정리할 수 있게 만들었다.

방향

코드와 게임 작업을 같이 보여주는 공간.

웹 프론트엔드 실험, Unity 프로젝트, 개발 메모, 도구 스크립트를 한 곳에서 이어서 보여줄 계획이다.

주요 영역

지금 이 사이트가 보여주기 좋은 세 가지.

01

3D 프로젝트

Unity 게임 플레이, 시스템 설계, 구현 과정을 정리해서 현재 작업 흐름을 보여준다.

02

웹 프로젝트

정적 페이지, 인터랙션 실험, 화면 구성 작업처럼 결과를 바로 확인할 수 있는 작업을 모은다.

03

개발 도구

자동화 스크립트, 유틸리티, 작업 문서를 묶어서 실제로 쓰는 개발 습관을 보여준다.

다음 단계

이제 프로젝트 링크와 실제 소개 문구를 채우면 바로 쓸 수 있다.

사이트 안에서 문구를 직접 고치고 저장하면 원본 `index.html`에 바로 반영되도록 편집 모드를 붙였다.

인증 대기

`?edit=1`에서만 열리는 편집 패널이다. GitHub 토큰을 확인한 뒤 네 계정일 때만 글씨 편집과 저장을 허용한다.

권장 권한: 이 저장소 하나만 선택한 fine-grained token + `Contents` 읽기/쓰기. 토큰은 현재 탭의 세션 저장소에만 유지된다.

편집 패널을 열려면 주소 뒤에 `?edit=1`을 붙여 접속하면 된다.
