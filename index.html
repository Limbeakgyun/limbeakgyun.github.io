GitHub Pages 시작점

임백균의 작업과 실험을 정리하는 개인 사이트.

이 페이지는 프로젝트, 실험, 앞으로 공개할 작업을 담아두기 위한 가벼운 홈 베이스다. 별도 빌드 없이 GitHub Pages에서 바로 동작하도록 구성했다.

GitHub 프로필 라이브 사이트

빠른 요약

  • 형식정적 HTML / CSS / JS
  • 배포GitHub Pages 메인 브랜치 루트
  • 초점프로젝트, 실험, 프로필

소개

빠르게 읽히고, 손쉽게 고칠 수 있게.

구조를 단순하게 유지해서 스크린샷, 저장소 링크, 소개 문구를 나중에 추가해도 사이트 전체를 다시 만들 필요가 없도록 잡아뒀다.

방향

코드와 게임 작업을 함께 담을 수 있게.

현재 구조는 소프트웨어 프로젝트, Unity 프로토타입, 개발 메모, 그리고 작은 쇼케이스 작업까지 무리 없이 담을 수 있다.

주요 영역

지금 이 사이트가 보여주기 좋은 세 가지.

01

3D 프로토타입

Unity 게임플레이 테스트, 레벨 실험, 짧은 버티컬 슬라이스를 정리하기 좋다.

02

웹 프로젝트

정적 페이지, 프론트엔드 작업, 작은 웹 앱처럼 깔끔한 소개가 필요한 결과물을 담는다.

03

개발 도구

자동화 스크립트, 유틸리티, 작업 흐름 실험처럼 문서화할 가치가 있는 도구를 보여준다.

다음 단계

이제 실제 저장소, 이미지, 연락처 정보로 채워 넣으면 된다.

구조는 이미 배포 가능한 상태다. 내용만 바꾸면 바로 개인 포트폴리오로 쓸 수 있다.
